## Rust gRPC service

This is a simple example of a gRPC service.

### Prerequisites

- [Docker](https://www.docker.com/products/docker-desktop/)

### Generate gRPC code

To generate the gRPC code, run:

    $ make codegen

### Build

To build the service, run:

    $ make build

### Run

To run the service:

    $ make run

### Test

To test the service:

    $ make test
